## 챕터 4: 자료구조

> **학습 목표:** Python의 4가지 핵심 자료구조를 이해하고, 각각의 특징과 사용 상황을 구분한다.

### 리스트 (`list`)
- [ ] 리스트 생성 (`[]`)
- [ ] 인덱싱 (양수/음수 인덱스)
- [ ] 슬라이싱 (`[start:stop:step]`)
- [ ] 주요 메서드: `append()`, `insert()`, `remove()`, `pop()`, `sort()`, `reverse()`, `len()`

### 튜플 (`tuple`)
- [ ] 튜플 생성 (`()`)
- [ ] 불변성 (immutable): 수정 불가
- [ ] 튜플 언패킹

### 딕셔너리 (`dict`)
- [ ] 딕셔너리 생성 (`{key: value}`)
- [ ] 키(key)로 값(value) 접근
- [ ] 주요 메서드: `keys()`, `values()`, `items()`, `get()`, `update()`, `pop()`

### 집합 (`set`)
- [ ] 집합 생성 (`set()`, `{}`)
- [ ] 중복 자동 제거
- [ ] 집합 연산: 합집합 `|`, 교집합 `&`, 차집합 `-`

### 자료구조 비교 정리
- [ ] list vs tuple: 가변 vs 불변
- [ ] list vs set: 순서 보장 vs 중복 제거
- [ ] dict: key-value 쌍 저장에 특화

## 챕터 11: 객체지향 프로그래밍 (OOP)

> **학습 목표:** 클래스와 객체 개념을 이해하고, 객체지향 프로그래밍의 핵심 원칙을 익힌다.

### 클래스와 객체
- [ ] 객체지향 프로그래밍(OOP)이란
- [ ] 클래스(Class): 설계도
- [ ] 객체/인스턴스(Object/Instance): 실제 만들어진 것
- [ ] `class` 키워드로 클래스 정의

### 생성자와 인스턴스 변수
- [ ] `__init__()` 생성자 메서드
- [ ] `self` 매개변수의 의미
- [ ] 인스턴스 변수 정의 및 접근

### 메서드
- [ ] 인스턴스 메서드
- [ ] 클래스 메서드 (`@classmethod`)
- [ ] 정적 메서드 (`@staticmethod`)

### 상속 (Inheritance)
- [ ] 부모 클래스와 자식 클래스
- [ ] `super()` 함수
- [ ] 메서드 오버라이딩 (재정의)

### OOP 핵심 개념
- [ ] 캡슐화 (Encapsulation): 데이터 은닉, `_` / `__` 네이밍
- [ ] 다형성 (Polymorphism): 같은 인터페이스, 다른 동작
- [ ] 특수 메서드 (매직 메서드): `__str__`, `__len__`, `__repr__`
